# Install git
```bash 
$ sudo apt-get update
```
#### Git
```bash 
$ sudo apt install git
```
#### Cloning
```bash 
$ sudo git clone <url>
```
#### Updation from server
```bash 
$ sudo git pull origin master
```
#### Pushing changes to server
```bash 
$ sudo git push origin master
```
#### Can be used without ```bash $ sudo```
